Odd-Job

Odd-Job is a job posting site that specializes in odd jobs. It allows users to find and post odd-jobs around the Greater Seattle Area. It uses AngularJS, HTML5, CSS3, GoogleMapsAPI, and Java Google Endpoints to display, manage, and store data.
